# Sales Agent Performance Dashboard (Power BI)

## Project Overview

This project presents a **Sales Agent Performance Dashboard** built using Power BI.
The dashboard analyzes sales data by comparing **planned targets with actual sales performance** and provides insights into agent productivity, branch contribution, and overall achievement.

The goal of this dashboard is to help managers and stakeholders **monitor sales performance, identify top-performing agents, and track target achievements across different branch groups.**

---

## Key Insights Provided

* Comparison of **Total Sales vs Total Plan**
* **Achievement Percentage** analysis
* **Branch-wise sales performance**
* **Role-based contribution to total sales**
* Identification of **Top 10 performing agents**
* Distribution of agents based on **status (Active / Suspended / Deleted)**

---

## Dashboard Features

* Interactive **slicers and filters** for branch groups
* KPI cards displaying **Total Sales, Total Plan, Active Agents, and Achievement %**
* Visual analysis through **bar charts, donut charts, and gauge charts**
* Identification of **high-performing agents and underperforming areas**

---

## How to Use

1. Open the `.pbix` file in Power BI Desktop.
2. Use the **Branch Group slicer** to filter the dashboard.
3. Analyze performance through different visualizations.
